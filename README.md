## 🚀 Key Features Implemented
- **Automated Entrance:** Splash Screen smoothly transitions to the Product List (Homepage) using an `After Delay` trigger.
- **Happy Path Navigation:** Seamless user flow from the Product List Screen to the Product Details Screen `On Click`.
- **Micro-interactions:** Interactive component with a `While Hovering` effect on the **Add to Cart** button to enhance the user experience.
- **No Dead Ends:** Back button navigation implemented to allow users to loop back to the homepage.
